# Aerealith AI Branding Package

Included assets:

- `logos/aerealith-ai-primary-poster.png` - source logo/poster image supplied for the package.
- `logos/aerealith-ai-mark-source.png` - cropped brand mark source used for icons.
- `logos/aerealith-ai-mark-concept.svg` - editable SVG concept mark.
- `icons/png/` - PNG icons from 16x16 through 512x512.
- `icons/ico/favicon.ico` and root `favicon.ico` - multi-size favicon.
- `icons/apple-touch-icon.png` - 180x180 app icon.
- `icons/android-chrome-192x192.png` and `icons/android-chrome-512x512.png` - web manifest icons.
- `tokens/aerealith-brand-tokens.json` - machine-readable brand tokens.
- `tokens/aerealith-brand.css` - CSS custom properties.
- `docs/aerealith-ai-brand-guide.docx` - brand guide document.
- `docs/aerealith-ai-color-palette-reference.png` - palette reference image.

Core positioning:

Aerealith AI is a secure digital companion and central command center for memory, automation, integrations, analytics, communities, and connected workflows.

Primary tagline:

A companion for a more connected world.

Support line:

Your central command center for your online presence.
